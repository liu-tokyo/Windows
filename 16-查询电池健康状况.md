## 查询电池健康状况



### 查询指令

- 命令提示符输入如下指令：

  ```cmd
  powercfg /batteryreport
  ```

  文件保存到如下位置：
  
  ```
  C:\Users\%username%\battery-report.html
  ```

### 健康分析

- 主要查询如下参数：

  | 参数名称             | 数值（例）         | 备注                                                         |
  | -------------------- | ------------------ | ------------------------------------------------------------ |
  | NAME                 | CP671396-01        | 这是电池的型号或标识号，通常用于区分不同的电池型号。         |
  | MANUFACTURER         | PTL                | 表示电池的制造商或供应商。在这种情况下，PTL 是电池的生产厂家。 |
  | SERIAL NUMBER        | 01B-Z140519000367Z | 电池的唯一序列号，用于跟踪和识别每个单独的电池。             |
  | CHEMISTRY            | LiON（锂离子）     | 表示电池使用的化学类型，在这种情况下是 **锂离子电池**，它是目前常见的一种电池类型，通常具有较高的能量密度和较长的使用寿命。 |
  | DESIGN CAPACITY      | 48,600 mWh         | 表示电池在出厂时的设计容量，单位是毫瓦时（mWh），表示电池理想情况下能够储存的能量。<br />这里是 48,600 毫瓦时，意味着它设计时可以储存 48.6 Wh（瓦时）的能量。 |
  | FULL CHARGE CAPACITY | 8,446 mWh          | 表示当前电池的实际充满电容量。8,446 毫瓦时比设计容量低很多，<br />说明电池的健康状态有所下降，充电能力已经不如新电池时那么强。 |
  | CYCLE COUNT          | 114                | 表示电池充放电的周期次数。一般来说，电池的循环次数越多，其容量会逐渐下降。<br />114 次循环是一个中等的次数，说明电池已经经历了不少充放电过程，但并不算特别多。 |

  总结来说，这个电池已经使用了一段时间，充电能力下降，容量只有原设计容量的约17%左右。虽然它仍然可以使用，但可能需要考虑更换电池，尤其是如果设备电池续航已显著下降。
