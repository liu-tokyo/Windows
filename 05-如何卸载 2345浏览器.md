# 如何卸载 2345浏览器

卸载 2345 浏览器（2345 王牌浏览器/加速浏览器）与卸载大多数 Windows 应用程序的方法类似。如果您在卸载过程中遇到残留或主页被锁定等问题，需要采取额外的步骤。

下面是详细的卸载步骤和清理建议：

## 🗑️ 步骤一：通过控制面板或设置卸载（标准方法）

### 选项 A: 通过 Windows 10/11 设置

1. 按下 `Win + I` 键打开 **“设置”**。
2. 点击 **“应用”**（Apps）。
3. 在应用列表中，找到 **“2345 王牌浏览器”** 或 **“2345 加速浏览器”**（名称可能有所不同）。
4. 点击该应用，然后点击 **“卸载”**。
5. 按照卸载程序弹出的提示完成卸载。

### 选项 B: 通过控制面板（适用于所有 Windows 版本）

1. 按下 `Win + R` 键，输入 `appwiz.cpl`，然后按回车键打开 **“程序和功能”**（或通过控制面板进入）。
2. 在列表中，找到 **“2345 王牌浏览器”** 或类似名称的程序。
3. 右键点击它，选择 **“卸载/更改”**。
4. 在卸载向导中，通常会有一个选项询问您**是否保留用户数据**或**彻底删除**。建议选择**彻底删除**。
5. 完成卸载。

## 🧹 步骤二：清理残留文件和注册表

卸载程序有时会留下一些文件或注册表项。您可以手动清理或使用第三方工具。

1. **清理残留文件夹**：
   - 卸载完成后，前往以下路径查看是否有残留的 **2345** 文件夹，如果有，请删除：
     - `C:\Program Files\`
     - `C:\Program Files (x86)\`
     - `C:\Users\您的用户名\AppData\Local\`
     - `C:\Users\您的用户名\AppData\Roaming\`
2. **检查启动项**：
   - 按下 `Ctrl + Shift + Esc` 打开**任务管理器**。
   - 切换到**“启动”**标签页，检查是否有任何与 2345 相关的项目，如果有，请**禁用**或**删除**。

## 🚫 步骤三：修复被锁定的浏览器主页

2345 浏览器最常见的问题是卸载后仍被强制修改或锁定其他浏览器的**主页**（所以感觉它很流氓、典型的国产流氓软件的特征）。

1. **修改浏览器快捷方式**：
   - 右键点击您常用的浏览器（如 Chrome 或 Edge）的**桌面快捷方式**。
   - 选择 **“属性”**。
   - 在 **“目标(Target)”** 栏中，检查路径末尾是否有类似于 `http://hao.2345.com` 的网址。
   - **删除**该网址，确保路径只到浏览器可执行文件 (`.exe` 结尾)，然后点击**确定**。
2. **在浏览器内重置主页**：
   - 打开您常用的浏览器，进入**设置**（Settings）。
   - 找到**“启动时”**（On startup）或**“主页”**（Homepage）设置。
   - 将启动页和主页修改为您希望的页面（例如：`about:blank` 或 `www.google.com`）。
