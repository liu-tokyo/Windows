# 远程桌面

## 1. 启用远程桌面

- 设置 → 系统 → 远程桌面

  启用远程桌面

  选择可以远程访问这台电脑的用户（注意：需要设置用户口令）

## 2. 设置自动登录

### 2.1 修改注册表

- 找到如下位置：

  ```
  HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\PasswordLess\Device
  ```

  把数值从 2 设置为 0 。

- 设置自动登录

  命令提示符输入如下指令：

  ```
  Netplwiz
  ```

  也可以使用如下指令：

  ```
  contral userpasswords2
  ```
