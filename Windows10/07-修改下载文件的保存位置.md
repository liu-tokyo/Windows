# 修改下载文件的保存位置

修改 Windows 10 中下载文件的保存位置主要有两种方法：**修改系统默认“下载”文件夹的位置**和**修改浏览器等应用程序的下载设置**。

## 一、修改系统默认“下载”文件夹位置（推荐）

这是最彻底的方法，可以更改大多数程序（包括文件资源管理器和大部分浏览器）默认引用的“下载”文件夹路径。

1. **创建新的目标文件夹：**
   - 打开“**此电脑**”（或文件资源管理器）。
   - 导航到您希望存放下载文件的**其他磁盘**（例如 D 盘）。
   - 在该磁盘下**新建一个文件夹**，例如命名为 `Downloads` 或 `我的下载`。
2. **打开“下载”文件夹属性：**
   - 打开“此电脑”或文件资源管理器。
   - 在左侧导航栏或主界面找到系统自带的“**下载**”文件夹。
   - **右键单击**该文件夹，选择“**属性**”。
3. **更改位置并移动文件：**
   - 在“下载 属性”窗口中，切换到“**位置**”选项卡。
   - 点击“**移动**”按钮。
   - 在弹出的窗口中，**选择**您在第 1 步中创建的新文件夹（例如 `D:\Downloads`），然后点击“**选择文件夹**”。
   - 点击“**应用**”。
   - 系统会提示您是否要将旧位置（通常是 C 盘）中的所有文件**移动**到新位置。建议选择“**是**”。
   - 等待文件移动完成后，点击“**确定**”关闭窗口。

## 二、修改浏览器或其他应用程序的下载设置

如果您只希望更改特定浏览器（如 Chrome、Edge、Firefox）的下载位置，可以在浏览器设置中单独修改。

### 1. Microsoft Edge 浏览器

1. 打开 **Edge** 浏览器。
2. 点击右上角的 **三个点 `...`** (设置及其它)，选择“**设置**”。
3. 在左侧导航栏选择“**下载**”。
4. 在“**位置**”下方，点击“**更改**”按钮，选择您希望保存下载文件的其他磁盘上的文件夹。
5. （可选）您可以开启“**每次下载时均询问保存位置**”的开关，以便每次下载时都手动选择位置。

### 2. Google Chrome 浏览器

1. 打开 **Chrome** 浏览器。
2. 点击右上角的 **三个点 `⋮`** (自定义及控制 Google Chrome)，选择“**设置**”。
3. 在左侧导航栏选择“**下载内容**”。
4. 在“**位置**”下方，点击“**更改**”按钮，选择您希望保存下载文件的其他磁盘上的文件夹。
5. （可选）您可以开启“**下载前询问每个文件的保存位置**”的开关。



## 💡 额外提示：更改新内容保存位置

您还可以通过 Windows **“设置”\**来更改\**新应用、文档、音乐等**内容的默认保存位置（这不会影响现有的“下载”文件夹）：

1. 按 `Win + I` 打开“**设置**”。
2. 选择“**系统**” > “**存储**”。
3. 找到并点击“**更改新内容的保存位置**”（或在“更多存储设置”下找到）。
4. 在弹出的窗口中，针对“**新应用将保存到**”、“**新文档将保存到**”等选项，选择您的目标磁盘（例如 D 盘）。
5. 点击“**应用**”。
