# 禁止指定的系统更新

在 Windows 10 中，如果你想**彻底掐死某一个特定的、不断报错或导致蓝屏的系统更新**，官方的“设置”界面是没有提供这个开关的。

但我们可以通过微软官方曾推出过的隐藏诊断工具 **`wushowhide`** 或者是功能更强大的极客工具 **WUmgr** 来精准实现这个需求。

以下是两种最有效、最干净的封杀方法：

## 方案一：使用微软官方隐藏工具（最推荐 ⭐⭐⭐⭐⭐）

微软官方曾发布过一个名为 **Show or hide updates (`wushowhide.diagcab`)** 的独立小工具，它的唯一功能就是把指定的更新从微软的排队队列中“隐藏”，从而达到禁止安装的目的。

### 操作步骤：

1. **下载工具：** 你可以复制下方微软官方的历史直连链接到浏览器中下载：

   Plaintext

   ```
   https://download.microsoft.com/download/f/2/2/f22d5fdb-59cd-4275-8c95-1be17bf70b21/wushowhide.diagcab
   ```

2. **运行扫描：** 双击打开下载好的 `wushowhide.diagcab`，点击 **下一步 (Next)**，工具会自动扫描你当前系统有哪些正在排队等待下载的更新。

3. **选择隐藏：** 扫描完成后，界面会出现两个选项，点击 **“Hide updates”（隐藏更新）**。

4. **勾选封杀：** 在弹出的列表中，**勾选你想要禁止的那一个特定更新补丁（例如：KB5066791）**，然后点击下一步。

5. **生效：** 提示修复成功后关闭工具。此时你再回到系统的“检查更新”里，这个补丁就会彻底消失。

> **如何反悔？** 如果以后你想重新装这个补丁，再次打开该工具，选择 **“Show hidden updates”（显示隐藏的更新）**，勾选它并恢复即可。

## 方案二：使用 WUmgr 独立更新管理器（最适合 Tiny10 等精简系统）

如果你使用的是 **Tiny10** 等精简版系统，方案一的官方工具可能会因为系统组件被阉割而导致闪退或报错。此时，使用第三方开源绿色工具 **WUmgr (Windows Update MiniTool)** 是最完美的硬核玩法。

### 操作步骤：

1. **运行软件：** 在网络上搜索并下载 **WUmgr**（它是一个单文件绿色版小工具），右键选择 **以管理员身份运行**。
2. **扫描队列：** 点击软件左侧工具栏顶部的 **“刷新” (Refresh)** 按钮（圆圈箭头图标），让它调用底层接口扫描当前系统正在排队的所有更新。
3. **一键隐藏：** * 在中间的列表中，找到你想要禁止的那个特定 KB 补丁。
   - 鼠标选中它，然后点击左侧工具栏上那个“眼睛带有斜杠”（或者锁形状）的图标，提示为 **Hide（隐藏）**。
4. **结果：** 被隐藏的补丁会变成灰色，并被归类到“隐藏的更新”标签页中，Windows 官方的更新程序将永远无法再检测和下载它。

## 💡 补充：如果这个讨厌的更新已经装上了怎么办？

如果你想禁止的那个更新**已经自动安装到了系统里**，导致系统卡顿或报错，你需要先把它卸载，再进行屏蔽。

1. 右键点击开始按钮，选择 **“命令提示符（管理员）”** 或 **“PowerShell（管理员）”**。

2. 输入以下命令强行卸载该补丁（以 KB5066791 为例，你可以换成你想删的补丁号）：

   DOS

   ```
   wusa /uninstall /kb:5066791 /quiet /norestart
   ```

3. 卸载完成后，**千万不要立刻重启电脑**。

4. 立刻打开上面的 **方案一** 或 **方案二** 的工具，将该补丁勾选为“隐藏”。

5. 屏蔽成功后，再重启电脑。这样系统重启后，就不会再次把它傻傻地下载回来了。
