# CTT 工具

> 在 Windows PowerShell 中非常著名的**快捷命令**，通常被称为 **“克里斯·蒂图斯 Windows 实用工具”（Chris Titus Tech Windows Utility，简称 CTT 工具）**。  
> 简单来说，它的作用是：**从网络上实时下载并运行一个功能极其强大的 Windows 系统优化、精简、管理和软件安装的图形化工具箱。**
>
> ```
> irm "https://christitus.com/win" | iex
> ```

## 1. Essential Tweaks（基础/核心优化）

与刚才的 Advanced Tweaks（高级优化）不同，**Essential Tweaks（基础/核心优化）** 是 Chris Titus 整个工具箱里**最安全、最推荐执行**的部分。

在这里，Chris 把那些“几乎对所有 Windows 11 用户都有利、且绝对不会破坏系统稳定”的优化项打包在了一起。它的核心逻辑是：**在不牺牲任何系统功能的前提下，砍掉微软的后台广告、停止无用的隐私数据收集、优化网络和硬盘读写。**

无论是你之前折腾的 Tiny11，还是普通的 Windows 11 专业版，这部分都可以**放心大胆地一键启用**。

### 1.1 为什么叫“Essential（必做）”？

因为勾选这部分的选项后，你的电脑通常会有以下直接的变化，且**零副作用**：

- **干掉系统各处的微软广告**：

  原版 Win11 在开始菜单、锁屏界面、甚至文件资源管理器里，经常会悄悄推送微软自己的产品广告或“使用建议”。Essential Tweaks 会把这些牛皮癣彻底清理干净。

- **优化系统的通知和弹窗**：

  关闭诸如“欢迎体验 Windows”、“完成设置您的设备”等每次大更新后都会跳出来的烦人全屏提示。

- **提升游戏和网络稳定性**：

  它会微调系统的网络堆栈，禁用诸如“网络带宽限制”等过时的默认设置，让游戏延迟更低，下载更稳定。

- **清理和腾出空间**：

  自动清理系统临时垃圾，并对固态硬盘（SSD）进行优化。

### 1.2 这里的核心功能有哪些？（你可以全部勾选）

虽然它支持“一键全选”，但我们可以看看它在后台具体帮你做了什么：

- **Disable Telemetry (Basic)** —— 基础隐私保护，阻止系统在后台不断向微软发送你的打字习惯、诊断日志等隐私数据，能省下一定的 CPU 占用。
- **Disable Consumer Features** —— 禁用所谓的“消费者特性”，也就是开始菜单里那些微软预装的第三方垃圾游戏图标（比如糖果传奇、抖音等广告位）。
- **Set Services to Manual** —— 把一些平时根本用不到、却一直在后台开机自启的 Windows 服务（例如传真服务、远程注册表修改等）改为“手动启动”，需要时才开，**开机速度和纯净内存会明显改善**。
- **Remove OneDrive** —— 如果你平时不用微软的网盘，它会帮你把这个极度吃内存、还经常自动同步桌面文件的启动项彻底剥离（这对配置不高的电脑非常友好）。

### 1.3 实操处理建议：直接点击 "Desktop" 或 "Laptop"

在 CTT 工具的 Tweaks 页面最上方，你会看到三个非常显眼的快捷预设按钮：

1. **如果你是台式机：** 直接点击 **Desktop** 按钮。
2. **如果你是笔记本：** 直接点击 **Laptop** 按钮（它会比台式机预设多保留一些电源管理和电池优化的设置）。

当你点击这两个按钮中的任意一个时，工具就会**自动把 Essential Tweaks 里面所有安全、推荐的选项全部勾选上**。

### 1.4 最后的运行步骤

确认勾选完 Essential Tweaks 的自动预设后，请看界面右侧的按钮区：



**1.创建还原点（安全保障）：**耗时约 30 秒。

在右侧蓝色按钮区，先点击 **Create Restore Point**（创建系统还原点）。这一步非常重要，万一以后你想恢复，随时可以一键倒流。

**2.执行优化：**耗时约 1-2 分钟。

点击最下方的 **Run Tweaks**（运行优化）。此时你会看到左侧的 PowerShell 蓝色背景窗口里开始滚动大量的代码绿字，这说明它正在修改注册表和清理系统。

**3.重启电脑：**关键步骤。

等待代码滚动停止，界面提示“Tweaks Finished”后，**手动重启一次电脑**。注册表和服务的修改只有在重启后才会彻底生效。



重启之后，你会发现系统的响应会轻快不少，后台的进程和内存占用也会有明显的下降！

## 2. Advanced Tweaks（高级优化

在 Chris Titus 的 CTT 工具中，**Advanced Tweaks（高级优化）** 是一个功能非常强大但同时也**带有一定风险**的区域。

不同于普通优化（Standard Tweaks）只修改一些无害的表面设置，Advanced Tweaks 会**直接深入系统注册表、强行剥离或禁用 Windows 的核心底层组件**。如果你正在使用的是本来就已经被精简过的 **Tiny11**，这部分的处理更需要格外小心，因为“二次精简”很容易导致系统彻底崩溃或某些功能永久失效。

### 2.1 Tiny11 系统

以下是针对 Tiny11 系统，对 Advanced Tweaks 里面最常见选项的处理建议和避坑指南：

#### 2.1.1 绝对不要碰的“红线”选项 ❌

在 Tiny11 上，以下几个选项**强烈建议保持默认（不要勾选）**：

- **Remove All UWP Apps（移除所有微软商店应用）**： Tiny11 本身就已经把自带的无用应用删得差不多了。如果在这里再次强行全盘切除，很容易导致 Windows 11 的 **Microsoft Store（应用商店）** 彻底损坏且无法修复，以后你想装任何来自商店的应用（或某些新版系统组件）都会报错。
- **Disable Windows Defender（禁用微软自带杀毒）**： Tiny11 有些版本已经对 Defender 做了精简或关闭处理。如果你在这里再次强行禁用，可能会导致系统设置里的安全中心页面“永久卡死”或“白屏”。如果你需要杀毒，建议保持现状，或者直接安装轻量级的第三方杀毒软件。
- **Remove Microsoft Edge（移除 Edge 浏览器）**： 在原版 Windows 上删 Edge 没问题，但在 Tiny11 上，很多系统内嵌的网页渲染组件（WebView2）和 Edge 深度绑定。强行剥离可能会导致某些依赖网页登录的软件、或者系统自带的某些小组件直接闪退。

#### 2.1.2 可以安全勾选的“实用”选项 （建议开启）

以下几项对降低内存占用、提升流畅度很有帮助，且不易和 Tiny11 发生冲突：

- **Disable Telemetry / Cortanna（禁用遥测和微软小娜）**： 虽然 Tiny11 已经清理过，但勾选此项可以确保注册表里的隐私数据上传通道被彻底锁死，还能顺便清理掉没用的语音助手残留，非常安全。
- **Delete Temporary Files（删除临时文件）**： 这只是纯粹的垃圾清理，每次使用工具时都可以勾选它，用来释放 C 盘空间。
- **Disable Hibernation（禁用休眠功能）**： 如果你使用的是台式机，或者笔记本不需要“休眠（Hibernate）”而只用“睡眠（Sleep）”，勾选它可以直接在 C 盘根目录**省下好几个 GB 到十几个 GB 的空间**（删除 `hiberfil.sys` 文件）。
- **Disable Sticky Keys（禁用粘滞键）**： 防止你在玩游戏或连击 `Shift` 键时，系统突然弹窗打断你。

#### 2.1.3 与你之前的“系统更新报错”相关的处理选项 🛠️

既然你是因为系统更新遇到错误才研究这个工具的，请特别留意 Advanced Tweaks 区域或 Updates 板块中关于 **Windows Update** 的选项：

- **Default Windows Update Settings（恢复默认更新设置）**： 如果你想修复更新报错，可以先勾选这一项，让工具尝试把被 Tiny11 或其他优化软件破坏的更新注册表项“校正”回微软默认状态。
- **Security Updates Only（仅安全更新）**： 如果你只想让 Tiny11 接收日常的漏洞补丁和驱动，不想让它升级大版本（防止系统体积爆满），这个选项是最佳选择。
- **Disable Windows Updates（彻底禁用更新）**： 如果你实在被刚才的报错折腾烦了，而且不打算让这个精简系统再升级，直接勾选此项。它会用最底层的权限把 Windows Update 彻底锁死，以后再也不会弹窗报错或在后台偷偷下载了。

> 💡 **黄金法则：** 在 Advanced Tweaks 里，**宁可少勾，也不要多勾**。建议每次只勾选 1-2 个你明确知道有什么用的选项，点击运行。如果系统表现正常，再倒回来开启其他的。切忌图省事一键全选。



### 2.2 普通 Windows11

如果是原装、未经过精简的 **普通 Windows 11 专业版（Pro）**，你折腾 Advanced Tweaks 的自由度就比在 Tiny11 上高得多了！因为你的系统底层组件是完整的，即使关掉某些功能，以后随时可以通过这个工具再勾选回来。

对于正常的 Win11 专业版，Advanced Tweaks 的核心目的通常是：**找回 Win10 的好用习惯、砍掉微软恶心人的广告/遥测、释放内存和硬盘空间**。

以下是针对普通 Win11 专业版的**实操推荐清单**，你可以对照着工具界面来勾选：

#### 2.2.1 强烈建议勾选的“神仙”选项（无痛提升体验）

这些选项能帮你关掉 Win11 那些饱受诟病的“特性”，而且绝不影响系统稳定：

- **Disable Telemetry（禁用遥测）**： 必选。切断微软在后台偷偷收集你电脑使用数据并上传的通道，能省下一定的 CPU 和网络带宽。
- **Disable Cortana（禁用小娜）**： 必选。微软自己都已经放弃这个语音助手了，关掉它能彻底释放它占用的后台内存。
- **Disable Wifi Sense（禁用 Wi-Fi 感知）**： 必选。这个功能会自动把你连接的 Wi-Fi 密码分享给你的联系人，存在隐私风险，果断关掉。
- **Disable Sticky Keys（禁用粘滞键）**： 必选。玩游戏或者快速连按 `Shift` 键时，再也不会弹出那个烦人的“是否开启粘滞键”的系统提示音了。
- **Disable Location Services（禁用位置服务）**： 如果你是用台式机，或者笔记本不需要天气、地图软件精准定位，关掉它可以减少后台消耗。

#### 2.2.2 根据个人习惯“二选一”或按需勾选的选项

- **Disable Hibernation（禁用休眠）**：
  - **如果是台式机：** 强烈建议勾选。台式机直接用“睡眠（Sleep）”即可。关掉休眠能瞬间帮你**在 C 盘根目录省下和内存一样大（比如 16GB 或 32GB）的硬盘空间**。
  - **如果是笔记本：** 保持不勾选（留着休眠），方便合上盖子长途携带时省电。
- **Remove Microsoft Edge（移除 Edge 浏览器）**：
  - **如果你平时只用 Chrome、Firefox：** 可以在这里放心地把它删掉，原版 Win11 即使删了 Edge 也会保留必要的网页渲染内核，不影响其他软件。
  - **如果你觉得 Edge 挺好用：** 别勾。
- **Classic Right-Click Menu（恢复经典右键菜单）**：
  - 如果你极其讨厌 Win11 那个需要点击“显示更多选项”才能看到全部功能的两层右键菜单，勾选它！它能一键帮你换回 Windows 10 那种传统的、一目了然的单层右键菜单。

#### 2.2.3 需要谨慎对待、甚至“不建议”勾选的选项 ⚠️

在标准专业版上，以下几项虽然不会导致系统崩溃，但会**明显改变使用功能**，勾选前请想清楚：

- **Remove All UWP Apps（移除所有微软商店应用）**： **不建议勾选**。这会实行“无差别大扫除”，把计算器、照片查看器、截图工具、甚至 **Microsoft Store（应用商店）** 本身全部强行卸载。一旦应用商店没了，以后想装回来极其麻烦。

  > **正确做法：** 别在高级优化里一键全删。可以去 CTT 工具的 **Install** 或 **Tweaks** 其他板块里，单独勾选卸载某些你不需要的自带软件（比如 OneDrive 或 微软小组件）。

- **Disable Windows Defender（禁用自带杀毒）**：

  - **如果你打算裸奔，或者已经安装了第三方杀毒软件：** 可以勾选。
  - **如果你依赖系统自带的安全防护：** 绝对不要勾。Windows Defender 虽然有时候敏感，但对普通用户来说是防范木马和勒索病毒最省心的防线。

#### 💡 动手前的最佳实践提示

点击右下角的 **Run Tweaks（运行优化）** 之前，工具界面通常会有一个 **Create Restore Point（创建系统还原点）** 的按钮。

**强烈建议在点运行前，先点一下它！** 这样万一优化完后你觉得某些功能用着不习惯，或者哪个软件报错了，你可以随时通过 Windows 的系统还原，一键秒回优化前的状态。



## 3. Customize Preferences（自定义偏好设置）

在 Chris Titus 的工具中，**Customize Preferences（自定义偏好设置）** 是一个非常人性化的板块。它不像刚才的 Advanced 涉及系统底层风险，也不像 Essential 是纯粹的性能优化，这一块**完全是用来满足你个人视觉审美和日常操作习惯的。**

这里面的选项没有“对与错”，只有“你喜不喜欢”。对于普通的 Windows 11 专业版，以下是针对这里最常见、最实用的选项的通俗拆解和处理建议：

### 3.1 强烈建议勾选的“提效神选项”（强烈推荐）

这些选项能帮你把 Windows 11 很多反人类的视觉和操作限制解开，让它用起来更像一个高效的生产力工具：

- **Show File Extensions（显示文件扩展名）**： **必选。** 微软默认会隐藏文件后缀名（比如把 `photo.jpg` 只显示为 `photo`）。开启这个后，你能一眼看出文件到底是图片、可执行程序（`.exe`）还是脚本，**不仅方便，还能大大降低误点木马病毒的风险。**
- **Show Hidden Files（显示隐藏文件）**： **推荐。** 搞电脑折腾配置、找AppData缓存文件时的必开选项。
- **Classic Right-Click Menu（经典右键菜单）**： **强烈推荐。** 恢复 Windows 10 传统的单层右键菜单。你再也不用为了点个“复制”或“重命名”，先右键、再痛苦地去点最下面的“显示更多选项”了。

### 3.2 视觉美化与布局选项（按个人喜好选择）

- **Dark Mode（深色模式） / Light Mode（浅色模式）**： 一键把整个系统（包括资源管理器、设置界面）切成全黑或全白。喜欢晚上看屏幕不刺眼的直接选 Dark Mode。
- **Snap Assist Flyout（贴靠助手弹窗）**： 当你把一个窗口拖到屏幕边缘时，系统会自动弹窗问你另一边要放哪个窗口。如果你觉得这个弹窗很烦，打扰你干活，可以在这里关闭它。
- **Disable Widget Service（禁用小组件）**： Windows 11 任务栏左下角那个经常闪烁、一碰就弹出大堆八卦新闻和天气预报的“小组件”面板。**如果你平时不看它，建议勾选禁用。** 它不仅在后台偷吃不少内存，还容易引起任务栏卡顿。

### 3.3 任务栏与开始菜单的“习惯找回”

Windows 11 的任务栏相比 Win10 做了巨大阉割，这里有几项可以帮你找回老习惯：

- **Taskbar Alignment (Left / Center)**：
  - **Center**（默认）：图标居中（像 Mac 风格）。
  - **Left**：把开始按钮和所有图标**一键移回左下角**，找回十几年来用 Windows 传统的肌肉记忆。
- **Disable Bing Search in Start Menu（禁用开始菜单的 Bing 搜索）**： **推荐勾选。** 平时我们按 `Win` 键想搜索电脑本地的一个软件或文件时，Win11 总是自作聪明地联网去加载 Bing 网页搜索结果，导致搜索框卡顿甚至白屏。关掉它后，**开始菜单的搜索会变成纯本地搜索，速度极快，瞬间秒开。**

### 🛠️ 实操建议

这部分的操作非常轻松，不需要创建还原点也完全没问题：

1. **按需勾选：** 扫视一遍列表，把你觉得看着顺眼、或者符合你平时习惯的条目勾选上。
2. **执行生效：** 点击右侧的 **Run Tweaks**（运行优化）。
3. **即时查看：** 这部分的修改大多数**不需要重启电脑**，你会发现任务栏图标瞬间移到了左边，或者右键菜单直接变了，效果立竿见影。如果你觉得哪项不习惯，再次打开工具把勾去掉，重新跑一次就能恢复原状。



## 4. Performance Plans（性能方案/电源计划）

在 Chris Titus 工具的 **Performance Plans（性能方案/电源计划）** 板块，它针对的是 Windows 的**电源管理管理机制**。

微软官方在系统里内置了不同的“电源计划”，通过控制 CPU 的运行频率、硬盘休眠时间、以及显卡的功耗，来平衡**电脑的性能**与**省电/发热**。

这个板块的作用，就是**帮你一键解锁 Windows 隐藏的顶级性能模式，或者优化笔记本的续航。**

### 4.1 两个按钮分别是什么作用？

在 Performance Plans 下方，你会看到两个核心按钮，它们的用途和适合的设备完全不同：

#### 4.1.1 按钮 1：Add Ultimate Performance（添加“卓越性能”模式）

- **这是什么：** 这是微软专为高端工作站和服务器设计的一个**隐藏顶级电源计划**，在普通的 Windows 11 设置里默认是被锁死、看不见的。点击这个按钮，工具会通过一条底层命令强制把这个模式“释放”出来。
- **实际效果：**
  - **CPU 零延迟：** 让你的 CPU 始终保持在较高的基础频率上，彻底取消 CPU 为了省电而频繁“降频、升频”的微小延迟。
  - **硬件不休眠：** 固态硬盘、PCIe 接口、无线网卡等设备被强制设置为“永不进入省电低功耗状态”，保证系统随时以绝对最高速度响应。
- **适用设备：** **台式机（Desktop）**。如果你经常玩大型游戏、剪辑 4K 视频、或者做代码编译，选这个。
- **⚠️ 笔记本警告：** **不建议笔记本在电池供电时用这个！** 它会极大地缩短电池续航，并显著增加发热量和风扇噪音。

#### 4.1.2 按钮 2：Enable Laptop Power Optimization（启用笔记本电源优化）

- **这是什么：** 这是 Chris Titus 团队针对移动设备（笔记本/掌机）定制的一套**省电与发热优化脚本**。
- **实际效果：**
  - **更聪明的休眠：** 优化系统在盖上笔记本屏幕、或者长时间不操作时的休眠（Modern Standby）机制，防止笔记本放在包里时在后台偷偷运行、发热耗电。
  - **激进的省电策略：** 在检测到你拔掉电源（使用电池）时，更快速地降低未使用的硬件功耗，从而**显著延长笔记本的电池续航时间**。
- **适用设备：** **笔记本电脑（Laptop）**。

### 4.2 实操建议：你该怎么选？

你可以根据你目前的电脑形态，选择最适合的操作流：

#### 4.2.1 情况 A：如果你用的是【台式机】

1. 点击 **Add Ultimate Performance** 按钮。
2. 此时点击后，系统表面可能没有太明显的提示，你需要去系统的传统设置里启用它：
   - 按 `Win + R` 输入 `control` 打开旧版**控制面板**。
   - 选择 **硬件和声音 $\rightarrow$ 电源选项**。
   - 你会发现多出来一个名为 **“卓越性能 (Ultimate Performance)”** 的勾选项，点击勾选它即可。

#### 4.2.2 情况 B：如果你用的是【笔记本电脑】

1. 点击 **Enable Laptop Power Optimization**。
2. 在电源选项中，建议保持微软默认的 **“平衡 (Balanced)”** 模式。配合这个工具的优化，系统会在你插电时跑满性能，拔掉电源时极致省电，这是最完美的平衡。
