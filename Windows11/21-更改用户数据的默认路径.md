# 更改用户数据的默认路径

将 **Windows 11 的用户数据**保存到其他磁盘（例如 D 盘）通常涉及到更改默认的**用户文件夹（如“文档”、“下载”、“图片”等）**的位置，以及更改 **新内容的默认保存位置**。

## 💻 更改用户文件夹位置

这是最常用的方法，用于移动您现有和将来在这些文件夹中的个人文件：

1. **打开文件资源管理器**：
   - 在左侧导航窗格中，点击“**此电脑**”。
   - 在右侧找到您的 **用户文件夹**（例如 **文档**、**下载**、**图片**、**视频**、**音乐** 等）。
2. **访问属性**：
   - **右键单击**您想要移动的文件夹（例如“文档”），然后选择“**属性**”。
3. **更改位置**：
   - 在弹出的窗口中，切换到“**位置**”选项卡。
   - 点击“**移动...**”按钮。
4. **选择新路径**：
   - 浏览到您想要保存数据的目标磁盘（例如 `D:` 盘）并选择或**创建一个新文件夹**（例如 `D:\用户\您的用户名\文档`）。
   - 点击“**选择文件夹**”。
5. **确认移动**：
   - 回到“位置”选项卡，路径已更新。点击“**确定**”或“**应用**”。
   - 系统会询问您是否要将**所有现有文件**从旧位置移动到新位置。请选择“**是**”。
   - 等待文件移动完成。

您需要对每个想要移动的用户文件夹（文档、下载、图片等）重复上述步骤。

※ *作业结束之后，需要重新启动电脑才能生效。*

## ⚙️ 更改新内容的默认保存位置

您可以设置让未来下载的文件、应用、文档等默认保存到非 C 盘的其他磁盘：

1. **打开设置**：
   - 按下 $\mathbf{Win} + \mathbf{I}$ 快捷键打开“**设置**”。
   - 选择“**系统**”。
2. **进入存储设置**：
   - 点击右侧的“**存储**”。
   - 点击“**高级存储设置**”（可能需要向下滚动）。
3. **更改保存位置**：
   - 点击“**保存新内容的保存位置**”或类似的选项。
4. **设置默认位置**：
   - 在这里，您可以更改以下几项的默认保存位置，将其改为您的目标磁盘（例如 D 盘）：
     - **新的应用将保存到**
     - **新的文档将保存到**
     - **新的音乐将保存到**
     - **新的图片和视频将保存到**
     - **新的电影和电视节目将保存到**

**请注意：**

- **程序文件**（如安装的应用程序）最好还是安装在 **默认位置** 或在安装时手动选择路径，直接移动程序文件可能会导致程序运行错误。
- 此方法不移动已存在于 C 盘的用户数据，仅对未来的新内容生效。

------

如果您的目的是为 **Microsoft Store 下载的应用** 节省 C 盘空间，您可以使用 Windows 自带的“**应用和功能**”设置中的“**移动**”功能来迁移它们：

1. 打开“**设置**” $\rightarrow$ “**应用**” $\rightarrow$ “**应用和功能**”。
2. 找到要移动的应用，点击右侧的“**三个点**” $\rightarrow$ 选择“**移动**”。
3. 选择目标磁盘（例如 D 盘）并点击“**移动**”。
