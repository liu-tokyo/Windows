# Brave 浏览器

## 一、Brave 介绍及安装

**Brave 浏览器完美支持 Windows 11。**

实际上，在 Windows 11 上运行 Brave 是一个非常棒的组合。Brave 是一款基于 **Chromium** 核心（和 Google Chrome、微软 Edge 同样的底层）开发的浏览器，所以它在 Windows 11 下的兼容性、速度和稳定性都极其出色。

在 Windows 11 上安装和使用 Brave，你可以享受到以下几个核心优势：

### 1. 为什么在 Windows 11 上用 Brave？

- **完全兼容所有 Chrome 插件：** 既然底层一样，你可以直接去 **Chrome 应用商店**（Chrome Web Store）下载任何你习惯使用的扩展插件（如密码管理器、划词翻译等），一键安装，完美运行。
- **开箱即用的超强广告拦截（Brave Shields）：** 相比于 Chrome 需要单独装 AdGuard 或 uBlock，Brave 原生自带了极为强悍的拦截器。它能直接过滤掉网页牛皮癣、视频片头广告以及后台的隐私追踪器。
- **内存占用可能比 Chrome 更低：** 因为它在底层就直接切断了各种广告脚本和追踪代码的加载，所以在 Windows 11 里运行很多大型网页时，它的内存（RAM）和 CPU 开销往往比原版 Chrome 还要轻量一些。

### 2. 如何在 Windows 11 上无缝安装？

**1.官方渠道下载:**约 1 分钟。

打开电脑自带的 Edge 浏览器，访问 Brave 的官方网站 `brave.com`。点击页面上显眼的 **Download Brave** 按钮，系统会自动识别并下载适合 Windows 11 64 位的安装引导文件（通常叫 `BraveBrowserSetup.exe`）。

**2.一键运行安装:**约 2 分钟。

双击运行下载好的 `.exe` 文件。Windows 11 会弹出用户账户控制（UAC）提示，点击“是”。接下来安装程序会自动联网下载完整组件并完成部署，期间不需要你进行任何复杂的下一步配置。

**3.一键导入旧数据:**首次开机。

安装完成后 Brave 会自动打开。在初次设置向导中，它会提示你**“从其他浏览器导入书签和设置”**。你可以一键把你之前在 Chrome 或 Edge 里的历史记录、收藏夹、甚至保存的密码直接同步过来，实现无缝切换。

**4.设为默认浏览器（可选）:**Windows 11 系统设置。

如果你想让它彻底代替 Edge，可以在 Brave 顶部的提示条中点击“设为默认”。或者在 Windows 11 系统中打开 **设置 $\rightarrow$ 应用 $\rightarrow$ 默认应用**，找到 Brave 并点击“设为默认值”。



> **💡 特别提醒：**
>
> 如果你的 Windows 11 是之前聊到的 **Tiny11 精简版** 或者是**老旧电脑**，Brave 同样是极佳的选择。因为 Tiny11 默认拔掉了 Edge 浏览器，你刚好可以用安装包非常干净的 Brave 来作为系统的主力网络浏览器。



## 二、Tiny11 如何 安装 Brave浏览器

由于 **Tiny11** 默认精简掉了微软的 Edge 浏览器，很多刚装完系统的人会发现电脑里“没有一个可以上网下载软件的浏览器”。

不用担心，在 Tiny11 上安装 Brave 浏览器非常简单，这里为你提供 **3 种最常用、最安全** 的安装方法，你可以根据自己当前的网络情况选择：

### 方法一：使用 Windows 11 自带的“微软商店”（最推荐、最简单）

虽然 Tiny11 精简了很多东西，但它通常完整保留了 **Microsoft Store（微软商店）**。

1. 点击屏幕底部的开始菜单，或者在任务栏上找到 **Microsoft Store**（蓝色带微软标志的购物袋图标）并打开。
2. 在顶部的搜索栏中输入 `Brave`。
3. 在搜索结果中找到 **Brave Browser**，点击 **“获取” (Get)** 或 **“安装” (Install)**。
4. 系统会自动完成下载和部署，完全不需要通过任何第三方浏览器。

### 方法二：使用 Windows 自带的命令行工具（极客推荐，一步到位）

Windows 11 内置了一个强大的官方包管理器叫 **Winget**，哪怕系统没有浏览器，只要有网络，就能通过一行命令直接从官方渠道抓取并安装 Brave。

1. 按下键盘上的 `Win + X` 键，在弹出的菜单中选择 **“终端” (Terminal)** 或 **“命令提示符” (CMD)**。

2. 在黑色的命令行窗口中，复制并粘贴以下命令，然后按回车：

   ```
   winget install Brave.Brave
   ```

3. 此时系统会提示你是否同意服务条款，按下键盘上的 `Y` 然后回车。 
4.  接下来你会看到进度条，系统会自动去 Brave 官网下载最新的稳定版并静默安装。完成后，桌面上就会直接出现 Brave 的图标。 

---

### 方法三：在宿主机/其他电脑上下载，用 U 盘拷贝（适合虚拟机或没联网的机器） 

如果你是在 **VMware 虚拟机**里跑 Tiny11，或者这台老电脑暂时连网不方便： 

1. 在你原本的正常电脑（宿主机）上，打开浏览器访问 Brave 官网 `brave.com`。 
2. 建议下载 **离线完整安装包（Standalone Installer）**，这样可以避免在 Tiny11 里因为网络问题导致在线安装卡死。   
   * *小技巧：访问 `[brave.com/download](https://brave.com/download)`，通常在页面下方或右侧会有“Standalone installer”或“Offline installer”的字样。* 
3. 下载得到一个几十兆或一百兆左右的 `.exe` 安装文件。 
4. 把这个文件拷贝到 **U 盘**里，然后插到 Tiny11 的电脑上（或者直接通过 VMware 的共享文件夹/拖拽功能复制进虚拟机）。 
5. 在 Tiny11 里双击运行这个 `.exe` 文件，即可直接完成离线安装。 

---

> **💡 安装后的贴心建议：** 

> 成功打开 Brave 后，它会默认自带一个叫 **Brave Rewards（Brave 奖励）** 和 **Brave Wallet（加密钱包）** 的功能，并且有时会展示一些官方广告。如果你不需要这些，可以在 Brave 的 **设置 $\rightarrow$ 缘由/奖励** 中把它们彻底关闭，让界面变得像 Tiny11 系统本身一样干净纯粹。
